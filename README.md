# Memory Game

Android memory game application where users test their memory by matching pairs of cards.  
The project focuses on basic game logic, user interaction, and UI handling.

---

## Features
- Classic memory card matching gameplay
- Interactive and user-friendly interface
- Restart game functionality
- Lightweight Android application

---

## Tech Stack
- Java
- Android Studio
- Android SDK
- Git & GitHub

---

## Getting Started
1. Clone the repository
2. Open the project in **Android Studio**
3. Sync Gradle files
4. Run the app on an emulator or physical device
